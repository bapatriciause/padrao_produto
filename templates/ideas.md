# Ideas Pool

This is the candidate pool for the clustering step. Each section is filled in by a different step of ideation. Provenance is structural (which section an idea sits in) plus an inline tag.

> **Workspace placement:** copy this file to `workspaces/<your-name>/ideas.md` and fill in there. Don't write to the template.

---

## How to enter an idea

Use this shape for every entry, in every section:

```
### [Idea title — 3–6 words]

- **Description:** [1–2 sentences in plain English. What is it?]
- **HMW answered:** [the exact HMW question this idea responds to]
- **Source problem statement:** [which PS this HMW traces to]
- **Tag:** [Perspective 1 / Perspective 2 / Perspective 3 / from opportunity area / pre-existing]
```

(For opportunity areas, replace "HMW answered" / "Source problem statement" with the original signal from `opportunity-areas.md`.)
