# Volaris AI Accelerator, Product Track

Your working repository for the AI Accelerator. All exercises happen during the workshop using the prompts and templates in this repo.

## Getting Started

1. Clone: `git clone <YOUR-VBU-REPO-URL>`
2. Read: `guides/github-and-repo-guide.md`
3. Open `prompts-v2/README.md` and follow the guided chapter flow. Start with `prompts-v2/00-onboarding.md`.
4. Start working
5. If you get stuck, check `troubleshooting/README.md` first.

## Saving Your Work

Don't worry about git commands. When you want to save your progress, just tell Claude in plain English:

- **"Save my work"** or **"back this up"** — Claude will commit and push for you.
- Claude will ask what you were working on (a few words is fine). That becomes the label so you can find the save later.

If you'd rather learn the git commands, see `guides/github-and-repo-guide.md`.

- If more than one person from your VBU is using this repo, create your own folder under `workspaces/` to keep things cleaner
- Don't edit `prompts-v2/`, `templates/`, `guides/`, or `troubleshooting/` — copy a file into your workspace if you need to adapt it
- Commit often with descriptive messages
- Ask a facilitator if you need help
